1 Multi Factor Authentication Multi-factor authentication is an electronic authentication method
in which a user is granted access to a website or application
only after successfully presenting two or more pieces of
evidence to an authentication mechanism.
  • In AWS Console ---> Secority, Identity & Compliance

  • Select IAM

  • Check for the Secority Alert

  • Select MFA

  • Activate MFA

  • Select Virtual MFA device ---> Continue

  • Show QR Code

  • Scan QR Code on phone Google Authenticator

  • Google Authenticator Start genrating MFA codes

  • Add first code MFA code 1

  • Add second code MFA code 2 ---> Click Assign MFA

  • Click on User on left side under Access Mangement

Add New User

  • Click on Add User

  • Add User Name*

  • Select AWS access type (one or both according to requirement)
    * Programmatic access
    * AWS Mangement Console access

  • Console password* (select one of the option)
    * Autogenerated password
    * Custom password

  • Select Require password reset
  • Next: Permissions
  • Add User to Group
  • Select Create Group
  • Add Group Name
  • Select Policy
  • Click Create Group
  • Add Tags
    * Key
    * Value
  • On next screen Click Create User
  • Either Download.csv or send email
  • Close

3 Create Role AWS roles are a secure way to grant permissions to entities that you trust
  • Click on Create Role

  • From Filter Policies select S3
  • Select Policy Name
  • Click Next
  • Click Next on Tab
  • Add Role Name*:
  • Click Create Role
