Storage Volumes that you can attach yo your EC2 instances
  • Types of EBS Volume:
      * General Purpose SSD(gp2)
      * Provisioned IOPS SSD (io1)
      * Provisioned IOPS SSD (io2)
      * Throughput Optimized HDD (st1)
      * Cold HDD (SC1)
  • Creating EBS volume:
      * In EC2 Dashboard select Volumes
      * Create Volume
      * Add Required information >>> EBS_1
  • Attach Volume to EC2 Instance
      * Select the volume
      * Click on Actions
      * Select attach Volume
      * From droupdown select instance
      * Then Click on Attach
